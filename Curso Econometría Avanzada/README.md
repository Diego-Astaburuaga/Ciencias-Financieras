# Curso Econometría Avanzada

Este curso aborda técnicas econométricas avanzadas aplicadas al análisis de fenómenos económicos y financieros. El foco está en modelos de variable dependiente limitada o discreta, tales como modelos Probit y Logit, que permiten analizar decisiones binarias o probabilidades de ocurrencia de eventos. Se estudia la teoría subyacente, la estimación por máxima verosimilitud y la interpretación de efectos marginales, con aplicaciones a datos reales del sistema financiero y previsional chileno.
