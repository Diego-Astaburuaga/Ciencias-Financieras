# Curso Gestión de Inversiones

Este curso aborda la teoría y práctica de la gestión de portafolios e inversiones financieras. Se estudia la teoría moderna de portafolios de Markowitz, estrategias de trading técnico (medias móviles), análisis de fondos mutuos, anomalías de calendario y estrategias cuantitativas de momentum. El énfasis está en la aplicación empírica de estos conceptos a mercados financieros reales, principalmente latinoamericanos y de criptomonedas.
