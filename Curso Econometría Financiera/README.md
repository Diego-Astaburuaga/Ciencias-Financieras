# Curso Econometría Financiera

Este curso aplica herramientas econométricas al análisis de datos y fenómenos propios de los mercados financieros. Se estudian metodologías como estudios de eventos (*event studies*), modelos de retornos anormales, CAPM, regresiones OLS y robustas (RLM), y métricas de correlación como Spearman y Kendall. El objetivo es desarrollar capacidad para analizar la eficiencia de mercado, el impacto de eventos corporativos sobre los precios de activos, y la dinámica de mercados emergentes como el chileno.
